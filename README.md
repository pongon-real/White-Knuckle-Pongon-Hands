# White-Knuckle-Pongon-Hands
A Mod for the indie climbing game White Knuckle that lets you use pongons hands.


HOW TO USE:

Step 1 - Download the Zip

Step 2 - Unzip the Folder to C:\Program Files (x86)\Steam\steamapps\common\White Knuckle\White Knuckle_Data\Mods\Cosmetics\Hands
NOTE: MAKE SURE ITS NOT TWO FOLDERS INSIDE EACHOTHER.

Step 3 - Enjoy!!
